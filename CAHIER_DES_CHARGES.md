## Mode Gamepad

- ON/OFF
- Mapping

- Mode de prise en compte des touches clavier
    - Simultané = les entrées clavier et manette sont enregistrées
    - Manette exclusif = les entrées clavier sont ignorées sur les touches mappées à une entrée manette, les autres entrées clavier sont enregistrées
    - Manette uniquement = aucune entrée clavier n'est enregistrée

## Futures fonctionnalités

- Toggle : Appuyer un fois pour maintenir la touche enfoncée, appuyer une seconde fois pour la relâcher
- Mod tap : Maintener la touche enfoncé pour une action, appuyer pour une autre action
