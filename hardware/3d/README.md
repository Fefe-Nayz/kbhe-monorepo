# 3D Models

Mechanical assets are split by lifecycle:

- `current/`: current 75HE assembly, case, PCB, plate, knob and keycap exports.
- `legacy/`: older experiments and generated/test exports kept for reference.

For each part, `.f3d`/`.f3z` files are Fusion source archives and `.step`/`.stl`
files are exchange/manufacturing exports.
